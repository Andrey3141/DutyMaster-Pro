## [v2.0.0] - 2026-03-06

### Added
- Полноценный редактор цветов интерфейса (frontend_color_editor.py):
  - Выбор любых элементов UI через рекурсивный обход виджетов
  - Предпросмотр изменений в реальном времени без блокировки интерфейса
  - Сохранение кастомных стилей в settings.json с поддержкой отмены
  - Редактирование: фон, текст, рамка, шрифт (семейство + размер)
- Интеллектуальная система тем (frontend_ui_widgets.py):
  - Три встроенные темы: light, dark, mixed (градиент)
  - Автоматическое применение пользовательских цветов поверх темы
  - Поддержка динамической смены темы без перезапуска приложения
- Расширенная аналитика расписания:
  - Вкладка «Анализ» с детальной информацией о качестве генерации
  - Визуализация распределения дежурств по статусам здоровья
  - Автоматическое выявление и отображение замечаний
- Полное сохранение состояния приложения:
  - Сохранение/загрузка: ученики, места, алгоритм, тема, фиксированные дни, custom_colors
  - Автосохранение после каждого значимого действия
  - Сброс к заводским настройкам с подтверждением
- Гибкая настройка фиксированных дней:
  - Визуальное выделение фиксированных дней (📌 + серый стиль)
  - Блокировка редактирования назначений в фиксированные дни
  - Сохранение выбора фиксированных дней между сессиями
- Оптимизация UI-рендеринга: использование QScrollArea и ленивой загрузки для больших расписаний
- Улучшена валидация входных данных: подробные сообщения об ошибках
- Добавлены подсказки (tooltips) ко всем кнопкам и элементам интерфейса

### Fixed
- Ошибка применения пользовательских цветов к виджетам с динамическими именами
- Утечка памяти при многократном открытии диалогов (очистка temporary_styles)
- Некорректное отображение статусов здоровья в таблице статистики
- Проблемы с кодировкой при экспорте расписания в файл (добавлен encoding='utf-8')

## [v1.0.0] - 2026-03-03

### Added
- Бэкенд (backend.py):
  - Три алгоритма генерации: Жадный, Назначений, Случайный
  - Система приоритетов по статусу здоровья
  - Валидация входных данных и оценка качества расписания
  - Фабрика тестовых данных DataFactory
- Фронтенд (frontend_main_app.py):
  - Главное окно на PyQt6 с вкладками: Расписание, Статистика, Анализ
  - Интерактивное отображение расписания с цветовым кодированием
  - Диалоги управления: ученики, места, дни, алгоритм, тема
- Генерация расписания дежурств на 1-7 дней
- Поддержка 6+ мест дежурства с настраиваемым количеством дежурных
- Учёт статуса здоровья: здоров / недавно болел / болеет
- Фиксация отдельных дней для ручного редактирования
- Экспорт расписания в текстовый файл
- Три темы оформления: светлая, тёмная, смешанная
- Современные виджеты: ModernButton, ModernCard
- Таблицы с сортировкой и цветовым выделением
- Индикатор прогресса генерации
- Подсказки и информативные сообщения об ошибках
