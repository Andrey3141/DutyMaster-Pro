# 🏗️ Структура проекта: DutyMaster Pro

```
DutyMaster/
├── 📄 README.md                 # Основная документация проекта
├── 📄 CHANGELOG.md             # История изменений (этот файл)
├── 📄 PROJECT_STRUCTURE.md     # Описание архитектуры (этот файл)
├── 📄 requirements.txt         # Зависимости Python
├── 📄 settings.json            # Пользовательские настройки (автогенерируется)
│
├── 🐍 **БЭКЕНД (Ядро системы)**
│   └── backend.py              # Логика генерации расписания
│       │
│       ├── 🔧 КОНФИГУРАЦИЯ
│       │   ├── DAYS_COUNT      # Количество дней в расписании
│       │   ├── PLACES_CONFIG   # Места и количество дежурных
│       │   ├── MAX_ATTEMPTS    # Макс. попыток генерации
│       │   └── DEBUG           # Флаг отладки
│       │
│       ├── 📦 DATA CLASSES
│       │   ├── DayStatus       # Статус ребёнка на конкретный день
│       │   ├── Child           # Информация об ученике
│       │   └── ScheduleResult  # Результат генерации + метрики
│       │
│       ├── 🧭 БАЗОВЫЙ ПЛАНИРОВЩИК
│       │   └── BaseScheduler
│       │       ├── _validate_input()      # Валидация данных
│       │       ├── calculate_target_distribution() # Целевое распределение
│       │       └── _evaluate_schedule()   # Оценка качества расписания
│       │
│       ├── 🤖 АЛГОРИТМЫ ГЕНЕРАЦИИ
│       │   ├── GreedyScheduler      # Жадный алгоритм (рекомендуется)
│       │   ├── AssignmentScheduler  # Задача о назначениях (точная оптимизация)
│       │   └── RandomScheduler      # Случайный перебор (для сравнения)
│       │
│       ├── 📊 ВИЗУАЛИЗАЦИЯ
│       │   └── ScheduleVisualizer
│       │       ├── print_configuration()  # Вывод настроек
│       │       ├── print_schedule()       # Форматированный вывод расписания
│       │       └── print_statistics()     # Статистика распределения
│       │
│       └── 🏭 ФАБРИКА ДАННЫХ
│           └── DataFactory
│               └── create_sample_children()  # Генерация тестовых данных
│
├── 🎨 **ФРОНТЕНД (PyQt6 GUI)**
│   │
│   ├── 🖥️ ГЛАВНОЕ ПРИЛОЖЕНИЕ
│   │   └── frontend_main_app.py
│   │       │
│   │       ├── 🪟 ОСНОВНОЙ КЛАСС
│   │       │   └── DutyScheduleApp(QMainWindow)
│   │       │       ├── setup_ui()              # Инициализация интерфейса
│   │       │       ├── apply_theme()           # Применение темы + custom_colors
│   │       │       ├── generate_schedule()     # Запуск генерации с прогресс-баром
│   │       │       ├── display_schedule_result() # Отображение результата
│   │       │       ├── display_statistics()    # Вкладка статистики
│   │       │       ├── display_analysis()      # Вкладка анализа
│   │       │       ├── export_schedule()       # Экспорт в .txt
│   │       │       └── load_schedule_file()    # Импорт с выбором фиксированных дней
│   │       │
│   │       ├── 🗂️ УПРАВЛЕНИЕ ДАННЫМИ
│   │       │   ├── load_settings() / save_settings()  # JSON-сериализация
│   │       │   ├── reset_to_default()                 # Сброс настроек
│   │       │   └── update_stats()                     # Обновление статистики
│   │       │
│   │       ├── 🗣️ ДИАЛОГИ НАСТРОЕК
│   │       │   ├── open_days_dialog()        # Настройка количества дней
│   │       │   ├── open_students_dialog()    # CRUD учеников
│   │       │   ├── open_places_dialog()      # Управление местами
│   │       │   ├── open_algorithm_dialog()   # Выбор алгоритма
│   │       │   ├── open_theme_dialog()       # Смена темы
│   │       │   └── open_app_settings()       # Общие настройки приложения
│   │       │
│   │       └── 🎨 ТЕМИЗАЦИЯ
│   │           └── Взаимодействие с AppTheme и custom_colors
│   │
│   ├── 🎨 РЕДАКТОР ЦВЕТОВ
│   │   └── frontend_color_editor.py
│   │       │
│   │       ├── 🧭 ОСНОВНОЙ ДИАЛОГ
│   │       │   └── ColorEditorDialog(QDialog)
│   │       │       ├── collect_ui_elements()      # Рекурсивный обход виджетов
│   │       │       ├── categorize_elements()      # Группировка по типам
│   │       │       ├── preview_changes()          # Временный предпросмотр
│   │       │       ├── apply_changes()            # Постоянное применение + сохранение
│   │       │       └── reset_temporary_changes()  # Отмена предпросмотра
│   │       │
│   │       ├── 🔍 СЕЛЕКТОР ЭЛЕМЕНТОВ
│   │       │   └── ElementSelectorWidget
│   │       │       ├── Категории: кнопки, текст, фон, рамки, таблицы...
│   │       │       ├── Мультивыбор элементов
│   │       │       └── Детальная информация по элементу (tooltip)
│   │       │
│   │       ├── 🎨 ПАНЕЛЬ НАСТРОЕК
│   │       │   ├── Выбор свойства: фон / текст / рамка / шрифт
│   │       │   ├── Палитра предустановленных цветов (20 цветов)
│   │       │   ├── Произвольный выбор через QColorDialog
│   │       │   └── Настройки шрифта: семейство + размер
│   │       │
│   │       └── 🔧 ПРИМЕНЕНИЕ СТИЛЕЙ
│   │           └── apply_property_to_widget()
│   │               ├── Regex-замена CSS-свойств в styleSheet
│   │               ├── Применение QFont для текстовых свойств
│   │               └── Обработка ошибок без падения приложения
│   │
│   ├── 🧩 КАСТОМНЫЕ ВИДЖЕТЫ
│   │   └── frontend_ui_widgets.py
│   │       │
│   │       ├── 🎨 СИСТЕМА ТЕМ
│   │       │   └── AppTheme
│   │       │       ├── light_theme()   # Светлая (классическая)
│   │       │       ├── dark_theme()    # Тёмная (для глаз)
│   │       │       ├── mixed_theme()   # Градиентная (премиум)
│   │       │       └── get_theme()     # Фабричный метод
│   │       │
│   │       ├── 🔘 БАЗОВЫЕ ВИДЖЕТЫ
│   │       │   ├── ModernButton  # Кнопка с иконкой + hover-эффекты
│   │       │   └── ModernCard    # Карточка с тенью и скруглением
│   │       │
│   │       ├── ⚙️ ДИАЛОГИ НАСТРОЕК
│   │       │   ├── SettingsDialog   # Настройка дней + индикатор фиксации
│   │       │   ├── StudentDialog    # Добавление/редактирование ученика
│   │       │   ├── PlacesDialog     # CRUD мест дежурства
│   │       │   └── AboutDialog      # Информация о разработчике
│   │       │
│   │       └── 🔐 УТИЛИТЫ
│   │           └── Вспомогательные функции валидации и форматирования
│
├── 📁 **ДАННЫЕ И РЕСУРСЫ**
│   ├── settings.json           # Пользовательские настройки (автогенерируется)
│   │   ├── theme: str          # Выбранная тема
│   │   ├── custom_bg_color: str|None  # Пользовательский фон
│   │   ├── custom_colors: dict # Кастомные стили элементов
│   │   ├── children_data: list # Список учеников с настройками
│   │   ├── places_config: dict # Конфигурация мест
│   │   ├── algorithm: str      # Выбранный алгоритм
│   │   └── fixed_days: list    # Фиксированные дни недели
│   │
│   └── (опционально) exports/  # Папка для экспортированных расписаний
│
└── 🚀 **ЗАПУСК**
    └── python frontend_main_app.py
        │
        ├── 1. Загрузка settings.json (если есть)
        ├── 2. Инициализация DutyScheduleApp
        ├── 3. Применение темы + custom_colors
        ├── 4. Загрузка/генерация примерных данных
        └── 5. Отображение главного окна
```
